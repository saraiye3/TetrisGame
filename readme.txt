A classic Tetris game built in C++ using object-oriented design principles. This project features an intuitive user interface,
 allowing players to control falling tetrominoes, rotate them, and clear lines by completing rows.
 
The game includes:

Grid Management: Efficiently handles the placement and movement of pieces on a 2D grid.

Collision Detection: Ensures pieces interact correctly with the grid and other pieces.

Line Clearing: Automatically detects and removes filled rows, increasing the player's score.

Bomb Feature: Introduces a unique gameplay element where players can use bombs to clear multiple lines at once.

AI Computer Player: Challenge yourself against an AI opponent that plays Tetris, providing a competitive experience.

Game Mechanics: Implements traditional Tetris rules, with increasing difficulty as more lines are cleared.

Score Tracking: Keeps track of player scores, enhancing competitiveness
